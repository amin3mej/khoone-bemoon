Stay at home is a non-profit campaign which encourages people to stay at home and draw themself.

### Run in dev mode 
1. Install dependencies: 
`yarn`

2. Run: 
`yarn start`


### Build
1. Install dependencies:
`yarn --production --frozen-lockfile`

2. Build next.js:
`yarn build`

3. Start the project:
`yarn start`
